# Financial Document Management System

## Tech Stack
- Backend: Spring Boot
- Frontend: React
- Database: H2

## Features
- Upload financial documents
- View all documents
- Search documents

## RAG (Semantic Search)
Currently implemented using keyword search.

In production:
- Documents → chunking
- Embeddings generated
- Stored in vector DB (Qdrant/FAISS)
- Semantic retrieval

## How to Run

### Backend
Run Spring Boot app on port 8081

### Frontend
cd frontend
npm run dev
